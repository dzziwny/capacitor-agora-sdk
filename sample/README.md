# STEPS TO RUN

# common
1. npm ci  
2. type agora appId in line 11 in app.component.ts  
3. npm run build  

# web start
1. uncomment line 5 in app.component.ts  
2. npm start  

# ios build
1. npx cap add ios  
2. npx cap open ios + run from XCode  

# android build
1. npx cap add android  
2. npx cap open android + run from Android Studio

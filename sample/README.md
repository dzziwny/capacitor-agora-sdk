# STEPS TO RUN

# common
npm ci  
type agora appId in line 11 in app.component.ts  
npm run build  

# web start
uncomment line 5 in app.component.ts  
npm start  

# ios build
npx cap add ios  
npx cap open ios + run from XCode  

# android build
npx cap add android  
npx cap open android + run from Android Studio
